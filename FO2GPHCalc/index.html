Hello! This is a calculator for FO2 that will tell you the (rough) expected Gold Per Hour of a mob. Here's a short explanation of how and what you need to input that isn't obvious:

The crit and dodge values should be as they are written in game. So dodge should be "13.49", and crit should be "48.39". No % signs. No 0.2349. The same goes for attack speed; 2.4, 3.0, etc.

"HP Regen" isn't really a value, and is something I'm still testing. Use whatever green number pops up above your head between mob fights. If you're using a self-healing buff, just set this to some random stupid value. The purpose of this variable was to get an idea of which mobs are more "afk"/brainless, and wouldn't need me to keep popping HP pots to stay alive, but I'm still not sure if I want it or not.

All stats used should be after buffing yourself, obviously.

GPH values are estimates. They assume a lot of unrealistic stuff - basically 0 downtime between clicking a new enemy after the previous one has died, no time spent going to the shop to sell the items, no time spent looting. Consider them "extreme upper bounds" and apply common sense; if a mob has an unrealistically high GPH but it's got something like 700 item slots needed, it's probably going to be worse than a mob with similar gph who needs only 300 slots, for example.
